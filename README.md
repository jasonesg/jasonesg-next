# Jason's Website

Built with **Next.js**, **TypeScript**, and **Tailwind CSS**. Deployed on **Vercel**.

Personal portfolio and blog of Jason Esguerra.

## Features
- **Fast Performance:** Statically generated pages for instant loading.
- **Blog:** Markdown-based blog posts with front matter support.
- **Analytics:** Integrated Google Analytics 4 tracking.
- **Responsive Design:** Clean, minimal aesthetic that works on all devices.

## Local Development

First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
